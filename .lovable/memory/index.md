# Project Memory

## Core
ResumeForge platform. Dark landing (#1a1a2e) with lime accent (#c8ff00). Supabase backend.
Auto-confirm email. First download free, second paid via Fapshi.
Never use custom Supabase projects — Lovable Cloud only.
EN/FR language toggle. No PDF-to-Word converter (removed).
