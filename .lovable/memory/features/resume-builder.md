The platform provides a resume builder with 10 distinct templates:
- Classic, Modern, Minimal, Bold, Creative, Executive (with photo)
- Formal, Academic (no photo, traditional)
- Tech (developer-focused dark sidebar)
- Elegant (warm tones, sophisticated)

Features:
- Profile photo upload via Supabase Storage
- Skill icon picker (85+ logos from Simple Icons CDN)
- References section
- Local CV formats: Cameroon 🇨🇲, Nigeria 🇳🇬, Canada 🇨🇦
- AI suggestions for internship and junior dev roles
- EN/FR language toggle
- PDF export via html2canvas + jsPDF
