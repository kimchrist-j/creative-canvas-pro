# ResumeForge

Modern, lightweight resume + AI platform with multi-language support, AI analysis, real voice feedback, and PowerPoint generation.

## Features
- 10 CV templates, 10 languages (EN/FR/ES/DE/PT/IT/AR/ZH/JA/RU)
- AI CV analyzer — score /20 + recruiter voice feedback (ElevenLabs)
- AI CV generator from a freeform description
- AI PowerPoint generator with 4 designer templates
- App / wireframe builder
- Local formats: Cameroon, Nigeria, Canada migration
- Free first download, paid thereafter via Fapshi

## Local dev
\`\`\`
npm install
npm run dev
\`\`\`

## Deploy on Render (one click)
Includes render.yaml — push to GitHub, then Render → New → Blueprint → connect repo. Build command, env vars, SPA rewrite are preconfigured. No white page on refresh.

## Deploy on Vercel
Includes vercel.json with SPA rewrites: \`vercel --prod\`.

## Build commands
| Task | Command |
|---|---|
| Install | npm install |
| Build | npm run build |
